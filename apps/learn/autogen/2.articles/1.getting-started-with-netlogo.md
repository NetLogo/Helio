# {{ title }}

NetLogo is an open-source multi-platform agent-based modeling environment that is available on all major desktop operating systems (MacOS, Linux, Windows). In case you are unable to install the desktop application (e.g., on a Chromebook), you can use NetLogo as a web application at [[NetLogo Web|https://www.netlogoweb.org]]. In addition, a new cousin of NetLogo, called [TurtleUniverse](https://turtlesim.com/), is available as a native mobile application on Apple iOS and Google Android.

If this is your first time using NetLogo, you can read below to learn about how to get started.

#### Table of Contents

## Installing NetLogo on your desktop computer
If you are able to do so, we highly recommend you to install the desktop version of NetLogo on your computer so that you can use the full feature-set of NetLogo without any limitations.

Installing NetLogo on your desktop computer is as easy as it gets:

- Visit [[https://www.netlogo.org/download]] address
- You will be presented a simple optional form to enter your name, organization, email address, and comments.
- The page should detect your operating system automatically. If it does not, you can manually download NetLogo for your operating system by clicking on the appropriate link [Windows](https://www.netlogo.org/downloads/windows), [MacOS](https://www.netlogo.org/downloads/macos), or [Linux](https://www.netlogo.org/downloads/linux).
- You will have to select the correct architecture for your operating system as well (e.g., 64-bit vs 32-bit). Most modern computers are 64-bit.
  - If you're using MacOS, you will be given the option to download either the Intel or Apple Silicon version of NetLogo. Please choose the correct version for your computer's processor. Don't know what Mac you're using? Figure it out [here](https://macreports.com/how-to-know-if-your-mac-has-apple-silicon-or-intel-chip/).
- Click the Download link with the correct architecture for your operating system.
- Once you download the file, each operating system has a slightly different installation procedure.
- Double click the downloaded file and follow the regular procedure of your operating system to install NetLogo.

## Using NetLogo Web from the browser or Chromebook
In some cases, you may not be able to install the desktop version of NetLogo on your computer. For example, you may have a Chromebook, a tablet, or a mobile phone. In other cases, you may be using a public computer that does not allow installing new applications. In such cases, we recommend you to use NetLogoWeb, which is a browser-based version of NetLogo that works on all modern web-browsers.

You can access NetLogo Web by visiting [[https://www.netlogoweb.org]] address from your web browser. NetLogo Web works on all major browsers, including Chrome, Firefox, Safari, and Edge.

NetLogo Web can also be very useful in sharing the models you created on NetLogo Desktop with a greater audience without requiring them to install any other software.

Note that NetLogoWeb does not have feature-parity with NetLogo Desktop mainly due to limitations imposed by web technologies. This means that some extensions may not run on NetLogoWeb, some interface elements may not be editable (e.g., plots), and even some primitives may not function as expected (e.g., [[display]], [[wait]]). However, a significant majority of NetLogo features should run seamlessly in NetLogoWeb and most of your models should run on NetLogo Web without need for any modification.

## Models Library
If you just installed NetLogo, or you visited [[NetLogo Web|https://www.netlogoweb.org]], but you are new to NetLogo and you are unsure where to start, chances are the Models Library is the place to go. This library includes numerous models developed by the NetLogo team, scientific experts, and educators all around the world. Each model in this library is carefully inspected by the NetLogo team. You can find models on a variety of topics under categories such as Biology, Computer Science, and Social Science.

On NetLogo Desktop, you can access the models library by clicking the File menu and clicking the Models Library item in this menu.

On NetLogoWeb, you can simply use the combined search bar & drop down list to find a model in the library.

The following video tutorial shows how to access the models in the Models Library:

<video src="videos/models-library.mp4" controls class="rounded-lg border border-gray-300"></video>

It is easy to run the library models in NetLogo. Almost all of them have a **setup** button and a **go** button. To run a model, first click the **setup** button and then click the **go** button.

- Each library model has extensive information in the Info tab that you can use to learn more about the model and how to use it.
- Each library model is open-source. That is, you can go to the Code Tab and read each one's code.
