# Dataset Thumbnails

This directory should contain thumbnail images for dataset resources.

## Current Requirements:

- **population-dataset-thumb.png**: Thumbnail image for UN World Population Dataset
  - Suggested: World map, population graph, or demographic visualization
  - Dimensions: 400x300px or similar aspect ratio
  - Format: PNG with transparency support

## Image Sourcing:

You can use:
- Data visualizations created from the dataset
- Creative Commons licensed images from Wikimedia Commons or UN materials
- Generated charts/graphs from the data
- Generated placeholder images from services like placeholder.com

## Note:

Images in the `autogen/images/` directory are automatically accessible via `images/` paths in the YAML files.
