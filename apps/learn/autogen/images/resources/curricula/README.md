# Resource Thumbnails

This directory should contain thumbnail images for curriculum resources.

## Current Requirements:

- **connected-chemistry-thumb.png**: Thumbnail image for Connected Chemistry curriculum
  - Suggested: Image showing molecular models or chemistry visualization
  - Dimensions: 400x300px or similar aspect ratio
  - Format: PNG with transparency support

## Image Sourcing:

You can use:
- Screenshots from the actual curriculum materials
- Creative Commons licensed images from Wikimedia Commons
- Generated placeholder images from services like placeholder.com
- Custom graphics created for the curriculum

## Note:

Images in the `autogen/images/` directory are automatically accessible via `images/` paths in the YAML files.
